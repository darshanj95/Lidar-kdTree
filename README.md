# Lidar-kdTree

Use KdTree Nearest Neighbor search on Point cloud data to cluster point clouds.

## Dependency:
1. PointCloud Library (https://pointclouds.org/downloads/) 

## Installation

```bash
$ git clone https://github.com/darshanj95/Lidar-kdTree.git
$ cd Lidar-kdTree
$ mkdir build
$ cd build
$ cmake ..
$ make 
$ ./pcd_read
```
